# angular-websocket

# Iniciar el servicio de node-websocket-server

Ejecutar los siguientes comandos:

- npm install
- npm run dev

# Iniciar el servicio de laravel-api

Ejecutar los siguientes comandos:

- composer install
- php artisan serve

# Iniciar el servicio de angular-websocket

Ejecutar los siguientes comandos:

- npm install
- ng serve
